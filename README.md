MediaPulse — Интеллектуальный агрегатор новостей
Описание проекта
MediaPulse — полнофункциональный веб-приложение для автоматического сбора, классификации и отображения новостей из нескольких источников. Приложение использует Machine Learning для автоматической категоризации статей и APScheduler для фоновой автоматизации.
